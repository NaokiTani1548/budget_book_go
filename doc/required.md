# 最強家計簿アプリ

## 目的
- 一般的な家計簿アプリは機能が少ない/多すぎる場合が多い
- 少ない場合は支出と収入の登録のみで使い勝手が悪かったり、記録するだけになって面白みがない
- 一方で大規模なものになると基本的には使わない機能が多かったり、画面・ボタンが多すぎる印象で簡潔さに掛ける
- これを踏まえて必要な機能を必要な分だけ搭載した家計簿アプリを作成する

## 機能要件

### 必須要件

| ログイン | 登録ユーザーがログインできる | 認証処理 | 
| ログアウト | セッションを終了する | | 
| ユーザー情報編集 | ユーザー情報を編集する | メールアドレス・PWなど | 
| データ分離 | ユーザーごとに家計簿データを分離する | マルチユーザー対応 |

### 希望要件
- 画像から支出・収入の自動登録（OCR + AI要約）
    - 例: レシート画像から金額・日付を自動抽出
    - 使用例: Tesseract OCR
- 銀行残高参照
- 銀行APIと連携し残高を取得
- AIによる評価と改善案
- 支出傾向の分析
- 節約提案


## 非機能要件
- 基本的には各操作が0.5sec以内に収める（多くは期待しない）
- 認証にJWTを使用
- ログインを記憶
- できるだけすぐに開くことができるように
